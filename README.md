# weather-prediction-ml-model-analysis
 Analysis of various ML algorithms for rainfall prediction on Austrailian Dataset

### run locally

```
flutter run --dart-define=BACKEND_API=http://127.0.0.1:33507/
```

```
python3 backend/main.py
```
